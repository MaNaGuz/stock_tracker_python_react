# Python (Flask) - React Stock Tracker project

## Purpose

Practice React, Flask and Docker integration creating a Stock (inventory)
Tracker for a small store

## Usage

### Prerequisites

Docker Engine

### Build and Run

```docker-compose build``` if previous build was created and changes are required to build
```docker-compose up``` will build and start up if it's the first time

## Comments

this specific configuration will create a persistent SQLite DB
